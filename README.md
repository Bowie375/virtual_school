# virtual school
---

## Introduction
A map of PKU.

## Prerequisites
1. python: 3.9
2. node: v18.18.0
3. vue: @vue/cli 5.0.8
4. vite: vite/6.2.5 win32-x64 node-v18.18.0

## Setup
1. Clone the repository
2. Install the repository using `pip install -e .`
3. Navigate to `src/vschool/frontend` and run `npm install`
